# 1 Контрольная Фронтэнд и Бекэнд разработа

В рамках 1КР реализовано 5 практических заданий: 

- 1ПР: Реализован UI(User Interface) с помощью SASS и LESS, для визуального отображения карточек товаров, чтобы пользователь мог получить визуальное представление о товаре и базовую информацию
- 2ПР: Реализован базовый сервер на node.js, с эндпоинтами реализующие правила CRUD (Create, Read, Update, Delate), для того чтобы далее сделать маршрутизатор, и после этого полноценное API, для взаимодействие с данными на сервере с фронтенда. 
- 3ПР: Работа с Postman для отправления запросов на сервер и получения результата, для того, чтобы понять как отправляются запросы на сервер что приходит в виде ответа, как можно взаимодействовать с разными внешними API для получения данных, которые могут пригодиться пользователя. 
- 4ПР: Реализован полноценный сайт с фронтенд и бекэнд составляющей, которые разбиты по разным http портам, при помощи Express JS и ReactJS, RestfullAPI. Для того, чтобы пользователь мог получить доступ к удаленю, редактированию и изменению товаров.
- 5ПР: Реализована OpenAPI при помощи swagger, где документация находится по адресу `http://localhos:3001/api-docs/` . Чтобы разработчтк мог получить доступ к понятной и простой документации и быстро разобраться в API

## Итоги первой контрольной работы Магазин Верхняя одежда
Итогом 1 КР стал сайт с полноценным функционалом добавления, удаления и редактированию приложений, с реализованной документацией, и UI. 
Который запускается с помощью `npm run start` и `node src/api/script.js`
